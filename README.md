# Lambda
Lambdas are being used to complete one of the following tasks
- Tax calculation - avatax
- Create new order in DynamoDb - create-order
- Update order previously created - update order
- Form data collection - forms
- Get inventory from S3 - inventory

Lambdas should only be edited and uploaded from within the repo.
